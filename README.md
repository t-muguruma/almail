# Petal Email Client

Petalは、名作メールソフト「AL-Mail」の使い勝手を継承しつつ、現代的な機能とマルチアカウント対応を備えた、Windows向けの軽量・高機能メールクライアントです。

## 🌟 主な特徴

### 1. AL-Mail 完全移行サポート
- **設定インポート**: `ALMAIL.INI` からサーバー設定を自動取得。
- **アドレス帳**: `ALMAIL.ADR` からニックネーム、氏名、メールアドレスをインポート。
- **メッセージ**: 過去の受信・送信・下書きデータをフォルダ構造を維持したまま移行。

### 2. インテリジェント・機能
- **Google セキュリティ通知解析**: 重要なアラート（デバイス、場所、時間）を専用カード形式で表示。
- **SPF/DKIM/DMARC 認証**: メールの信頼性をリアルタイムで解析し、なりすましを警告。

### 3. モダンなメール体験
- マルチアカウント対応と統合受信箱。
- 送信済み・下書き・ゴミ箱の各フォルダ管理。
- 画像プレビュー機能（添付画像の本文末尾表示）。

## 🛠️ 技術スタック
- **Language**: Python 3.12+
- **UI**: Tkinter / TkinterWeb
- **Database**: SQLite3

## 📦 ビルド手順

ビルドを行うには、プロジェクトルートで以下のバッチファイルを実行してください。

```bash
build_app.bat
```

実行される工程：
1. アイコンの生成 (`Petal_icon.png` -> `Petal_icon.ico`)
2. PyInstaller による実行ファイルの生成
3. `move_exe.bat` によるファイルの配置
4. Inno Setup によるインストーラー作成

生成されたインストーラーは `W:\myProjects\Petal` フォルダ（バイナリ配布用リポジトリ）に出力されます。
